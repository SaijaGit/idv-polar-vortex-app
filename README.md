# Polar vortex anomalies

Final project for the Interactive Data Visualization course 2026.

This project is an interactive visualization for exploring the strength of the Arctic polar vortex and its relationship with winter temperatures in the Northern Hemisphere during winters 1975–2025.

The visualization combines a polar vortex strength timeline with an interactive temperature map. Users can compare different winters, examine official sudden stratospheric warming (SSW) events, and view either actual temperatures or temperature anomalies on different dates.


The project is implemented as an interactive Dash application using Plotly visualizations and ERA5 climate reanalysis data.

Run locally:

```
pip install -r requirements.txt
python app.py
```
